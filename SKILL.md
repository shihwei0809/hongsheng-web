# hongsheng-web 專案管理技能

本專案是「hongsheng-web」的實作專案。

## 🛠️ 環境依賴需求
1. **Node.js (npm)**：用於執行網頁伺服器或管理套件依賴。

## 🚀 AI 助理執行指引
1. **環境檢查**：
   - 檢查系統中是否已安裝相關依賴。
   - 若未安裝，請建議使用者執行 `powershell -ExecutionPolicy Bypass -File setup_env.ps1`。
2. **初始化與執行**：
   - 本專案可以直接雙擊 HTML 開啟，或使用本機 HTTP 伺服器託管運行。
